Introduce

소프트웨어과 전공 동아리인 RG는
게임 개발 전문 동아리입니다.
Unity 등의 사용 엔진과 동아리 자체 엔진을
이용해 직접 개발하고
다양한 디자인 도구를
활용해 게임을 꾸며나갑니다.

Developer

C, C++등 다양한 프로그래밍 언어를 공부합니다
Unity와 같은 상용엔진과 DirectX 기반의
동아리 자체엔진을 학습해 다양한
플랫폼에서 실행할 수 있는 게임들을 개발합니다

Designer

3D Max, Photoshop, illustrator, SAI등의
디자인 툴을 이용하여
3D 모델링, 게임 원화, 픽셀 아트(도트) 등
폭 넓은 게임 그래픽 디자인을 공부합니다

Curriculum

2020

Developer

Designer

C

많은 언어의 기초가 되는 C언어를 통해 프로그래밍의 기본을 학습하며 기초 역량을 기릅니다

Photoshop

도트 그래픽의 기초인 툴 세팅과 케릭터를 만들고 간단한 모션 만들기를 배우게 됩니다

UI/UX

게임 디자인의 기본인 UI/UX 수업을 이론으로 접근해 게임디자인의 이해를 돕고 사용자를 위한 디자인을 공부합니다

Console Game

C언어를 이용해 윈도우 콘솔 창에서 텍스트로 이뤄진 게임을 제작하며 게임의 구조와 동작을 이해합니다

After Effect

After Effect를 이용하여 게임에 들어가는 다양한 이펙트를 직접 제작합니다

Unity

사용 엔진을 사용하며 본격적인 게임 제작을 배우며 코드 작성뿐만이 아닌 그래픽 요소를 다룹니다

C++

객체지향 언어인 C++을 통해 클래스 구조를 이해하고 RG 엔진을 위한 기초를 다집니다

Magica Voxel

3D 도트를 만들 수 있는 프로그램으로 아기자기하고 귀여운 그래픽을 만들어 봅니다

3Ds Max / Blender / Zbrush

3D 캐릭터나 오브젝트를 모델링부터 질감과 세부적인 표현을 표현하는 걸 배우게 됩니다

RG Engine

자체 엔진을 사용해보면서 게임의 구조를 더 깊게 알아보고 엔진을 이해합니다

Substance Paint

노멀맵, 라이트맵 등 3d 모델링 후 uv 작업과 동시에 텍스쳐링을 하는 마무리 단계를 공부합니다
